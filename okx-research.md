# OKX Agent Trade Kit 活动研究

## 活动核心
- OKX正式推出Agent Trade Kit
- 业内首个为AI Agent和专业交易者同时设计的开源交易工具集
- 83个工具覆盖完整交易生命周期
- 支持现货、合约、期权、算法单、Bot等全品类交易
- MCP + CLI双入口
- 兼容所有MCP客户端：OpenClaw、Claude、Cursor、OpenCode、Codex
