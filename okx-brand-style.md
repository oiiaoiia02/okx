# OKX Agent Trade Kit 品牌视觉风格

## 配色方案
- 背景：纯黑 #000000 / #0A0A0A
- 文字：纯白 #FFFFFF
- 强调色：亮绿 #00FF88 / 荧光绿 (OKX品牌绿)
- 次级文字：灰色 #888888 / #AAAAAA
- 卡片背景：深灰 #1A1A1A / #111111，带微妙边框
- 代码块：深色终端风格
- 标签：深色背景 + 浅色文字

## 布局特征
- 全黑背景，极简
- Hero区域：大标题 + 彩色渐变球体背景
- 终端模拟器展示命令行交互
- 客户端Logo横排展示（OpenClaw/Claude/Cursor/OpenCode/Codex）
- 卡片式布局，深灰背景，圆角
- 标签系统（#AI-Native #Claude/GPT等）
- 大量留白，内容密度适中

## 字体
- 标题：大号粗体，可能是自定义字体
- 正文：清晰的无衬线字体
- 代码：等宽字体（终端风格）

## 交互元素
- "View on GitHub" / "Read the docs" 按钮 - 白色描边风格
- 代码复制按钮
- FAQ手风琴展开
- 终端动画效果

## 关键设计原则
1. 暗色主题为主（OKX官方就是暗色）
2. 绿色作为品牌强调色
3. 终端/代码风格突出技术感
4. 卡片式模块化布局
5. 大标题+副标题的层级结构
